## Built With

- Zig `0.14.0`
- See [`zig.mod`](./zig.mod) and [`zigmod.lock`](./zigmod.lock)

### Build from Source

```
$ git clone https://github.com/nektro/zigmod
$ cd zigmod
$ zig build -j1
```
